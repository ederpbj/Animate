## Iniciando:

git::https://github.com/ederpbj/Animate

>Clone

    git clone https://github.com/ederpbj/Animate.git

>Criar projeto

    react-native init Animate

>Corrigindo erros

    npm audit fix
    npm i --save-dev jetifier
    npx jetify

>Atualizar yarn

    npm i -g yarn

>Resetar cach

    react-native start --reset-cache
